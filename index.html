How it worksทำงานยังไง

Copy → warp → paste ✨ก๊อป → วาร์ป → วาง ✨

clipwarp watches your clipboard. When an image lands on it, it saves a PNG and puts that file's path on the clipboard as text — which Claude Code auto-attaches.clipwarp คอยดู clipboard พอมีรูปเข้ามา มันจะเซฟเป็น PNG แล้วเอา path ของไฟล์ใส่ clipboard เป็นข้อความ — ซึ่ง Claude Code จะแนบรูปให้เองอัตโนมัติ

Commandsคำสั่ง

Everything you can typeคำสั่งทั้งหมด

clipwarp watchStart the background auto-converter.เริ่มตัวแปลงอัตโนมัติเบื้องหลัง
clipwarp autostartStart the watcher automatically at every login.ให้ watcher เริ่มเองทุกครั้งที่ล็อกอิน
clipwarp statusIs the watcher running? Is autostart on?เช็กว่า watcher ทำงานอยู่ไหม + autostart เปิดอยู่ไหม
clipwarp stopStop the watcher.หยุด watcher
clipwarp unautostartRemove the login autostart.เอา autostart ตอนล็อกอินออก
cwShort alias — convert the current clipboard image once, then Ctrl+V.ตัวย่อ — แปลงรูปใน clipboard ครั้งเดียว แล้ว Ctrl+V
💡

Not using Claude Code right now? Turn the watcher off.ไม่ได้ใช้ Claude Code อยู่? ปิด watcher ก่อน

While clipwarp watch is running, every copied image also carries its file path as text. Apps that accept images still paste the image — but a plain text box will receive the path instead (e.g. C:\Users\you\.claude\pasted-images\clip-….png). When you're done with Claude Code, run clipwarp stop (and clipwarp unautostart to stop it launching at login). ระหว่างที่ clipwarp watch ทำงาน ทุกครั้งที่ก๊อปรูป clipboard จะพก path ของไฟล์เป็นข้อความไปด้วย แอปที่รับรูปได้จะวางเป็นรูปปกติ — แต่ช่องที่รับเฉพาะ ข้อความ จะได้ path มาแทน (เช่น C:\Users\you\.claude\pasted-images\clip-….png) พอเลิกใช้ Claude Code ให้รัน clipwarp stop (และ clipwarp unautostart เพื่อไม่ให้เปิดเองตอนล็อกอิน)

FAQ

Questions & answersคำถามที่พบบ่อย

Why doesn't Ctrl+V image paste work in Claude Code on Windows?ทำไม Ctrl+V วางรูปใน Claude Code บน Windows ไม่ได้?

Claude Code's terminal UI on native Windows can't read raw bitmaps from the clipboard, and Alt+V is WSL-only. Pasting a file path as text is the reliable route — clipwarp automates it.ตัว UI เทอร์มินัลของ Claude Code บน Windows แท้ ๆ อ่าน bitmap จาก clipboard ไม่ได้ และ Alt+V ใช้ได้แค่บน WSL วิธีที่เวิร์กคือวาง path ของไฟล์เป็นข้อความ — clipwarp ทำให้อัตโนมัติ

How do I paste a screenshot into Claude Code?วางสกรีนช็อตลง Claude Code ยังไง?

With the watcher running, take the screenshot (Win+Shift+S, PrtScn, Lightshot…), then press Ctrl+V in Claude Code. Without the watcher, run cw after the screenshot, then Ctrl+V.เปิด watcher ไว้ แล้วจับภาพ (Win+Shift+S, PrtScn, Lightshot…) จากนั้นกด Ctrl+V ใน Claude Code ถ้าไม่เปิด watcher ให้รัน cw หลังจับภาพ แล้วค่อย Ctrl+V

Does it work with WSL?ใช้กับ WSL ได้ไหม?

Under WSL, Claude Code's own Alt+V usually works. clipwarp targets native Windows (Windows Terminal, PowerShell, cmd, VS Code terminal), where nothing else does.บน WSL ปกติ Alt+V ของ Claude Code ใช้ได้อยู่แล้ว clipwarp ทำมาเพื่อ Windows แท้ (Windows Terminal, PowerShell, cmd, เทอร์มินัล VS Code) ที่ไม่มีอะไรใช้ได้

Is my screenshot uploaded anywhere?รูปถูกอัปโหลดไปไหนไหม?

No. clipwarp is a local PowerShell script — the image is written to a folder in your user profile and its path is put on your clipboard. Nothing touches the network.ไม่ clipwarp เป็นสคริปต์ PowerShell ในเครื่อง — รูปถูกเซฟลงโฟลเดอร์ในโปรไฟล์ของคุณ แล้วเอา path ใส่ clipboard เท่านั้น ไม่มีอะไรออกเน็ต

Which screenshot tools are supported?รองรับเครื่องมือจับภาพตัวไหนบ้าง?

Anything that puts an image on the clipboard: Snipping Tool, Win+Shift+S, PrtScn, Lightshot, ShareX, Greenshot, browser “Copy image”, Explorer file copies, and more.อะไรก็ได้ที่ใส่รูปลง clipboard: Snipping Tool, Win+Shift+S, PrtScn, Lightshot, ShareX, Greenshot, “Copy image” จากเบราว์เซอร์, ก๊อปไฟล์จาก Explorer และอื่น ๆ

What are the requirements?ต้องใช้อะไรบ้าง?

Windows 10 / 11, Windows PowerShell 5.1 (preinstalled) or PowerShell 7, and Claude Code running in any native Windows terminal.Windows 10 / 11, Windows PowerShell 5.1 (มีมาให้อยู่แล้ว) หรือ PowerShell 7 และ Claude Code ที่รันในเทอร์มินัล Windows แท้
