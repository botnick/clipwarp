Questionsคำถามที่พบบ่อย

Why doesn't Ctrl+V image paste work in Claude Code on Windows?ทำไม Ctrl+V วางรูปใน Claude Code บน Windows ไม่ได้+
Claude Code's terminal UI on native Windows can't read raw bitmaps from the clipboard, and Alt+V is WSL-only. Pasting a file path as text is the reliable route — clipwarp automates it.ตัว UI เทอร์มินัลของ Claude Code บน Windows แท้ อ่าน bitmap จาก clipboard ไม่ได้ และ Alt+V ใช้ได้แค่บน WSL วิธีที่เวิร์กคือวาง path เป็นข้อความ — clipwarp ทำให้อัตโนมัติ
How do I paste a screenshot into Claude Code?วางสกรีนช็อตลง Claude Code ยังไง+
With the watcher running (clipwarp watch), take a screenshot with Win+Shift+S, then press Ctrl+V in Claude Code. Without the watcher, run cw after the screenshot, then Ctrl+V.เปิด watcher ไว้ (clipwarp watch) แล้วจับภาพด้วย Win+Shift+S จากนั้นกด Ctrl+V ใน Claude Code ถ้าไม่เปิด watcher ให้รัน cw หลังจับภาพ แล้ว Ctrl+V
Does it work with WSL?ใช้กับ WSL ได้ไหม+
Under WSL, Claude Code's own Alt+V usually works. clipwarp targets native Windows — Windows Terminal, PowerShell, cmd, VS Code terminal.บน WSL ปกติ Alt+V ของ Claude Code ใช้ได้อยู่แล้ว clipwarp ทำมาเพื่อ Windows แท้
Is my screenshot uploaded anywhere?รูปถูกอัปโหลดไปไหนไหม+
No. clipwarp is a local PowerShell script. The image stays in your user profile and only its path is put on your clipboard. Nothing touches the network.ไม่ clipwarp เป็นสคริปต์ในเครื่อง รูปอยู่ในโปรไฟล์ของคุณ วางแค่ path ลง clipboard ไม่ออกเน็ต
Will it clutter my disk?รูปจะรกเครื่องไหม+
No. Saved PNGs go to %USERPROFILE%\.claude\pasted-images, and anything older than 7 days is cleaned up automatically — the folder never grows unbounded.ไม่ รูปที่เซฟไปเก็บที่ %USERPROFILE%\.claude\pasted-images และไฟล์ที่เก่ากว่า 7 วันจะถูกลบอัตโนมัติ — โฟลเดอร์ไม่บวมไม่รกเครื่อง
