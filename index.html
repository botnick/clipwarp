How it works

Copy → warp → paste ✨

clipwarp watches your clipboard. The moment an image lands on it, it saves a PNG and quietly puts that file's path on the clipboard as text — which Claude Code happily attaches.

1
✂️

Copy any image

Win+Shift+S, Snipping Tool, Lightshot, ShareX, or “Copy image” in a browser.

2
🍑

clipwarp warps it

Saves a PNG and swaps the clipboard to its path — automatically, in the background.

3
🎉

Ctrl+V in Claude

Press paste in Claude Code and the screenshot attaches. No extra steps.
